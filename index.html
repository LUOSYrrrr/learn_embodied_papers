关于本项目

这是一个个人学习笔记仓库，记录具身智能（Embodied AI）方向的论文精读过程。 每篇笔记包含：论文核心贡献、架构拆解、关键公式推导、以及与实际工程/研究的联系。 使用中文写作，英文专业术语保留对照，方便查阅原文。

如有错误或建议，欢迎通过 GitHub Issues 指出。
